# Создание иконок для PWA

## Быстрый способ

1. Откройте [Favicon Generator](https://favicon.io/favicon-generator/)
2. Создайте простую иконку с текстом "Г22" или используйте иконку шапочки
3. Загрузите иконку и скачайте в разных размерах
4. Или используйте [RealFaviconGenerator](https://realfavicongenerator.net/) для автоматической генерации всех размеров

## Ручной способ

1. Откройте `public/icon.svg` в любом графическом редакторе (Inkscape, Figma, Adobe Illustrator)
2. Экспортируйте как PNG:
   - 192x192 px → `icon-192.png`
   - 512x512 px → `icon-512.png`
3. Поместите файлы в папку `public/`

## Требования к иконкам

- Формат: PNG
- Размеры: 192x192 и 512x512 пикселей
- Фон: может быть прозрачным или цветным
- Рекомендуется: квадратные, без скругления (оно применяется автоматически)

## Альтернатива

Если нет возможности создать иконки сейчас, приложение все равно будет работать как PWA, просто без красивых иконок. Плагин создаст базовые иконки автоматически.
